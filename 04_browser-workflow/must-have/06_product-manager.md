### Task 1: Write User Stories

**Optimized Prompt Pattern:**
```
Generate user stories for:
[describe feature]

Context:
- User persona: [type of user]
- Goal: [what they want to achieve]
- Problem: [current pain point]

Format:
As a [persona]
I want [capability]
So that [benefit]

Include acceptance criteria and edge cases
```

**Token savings:** ~65% (product approach pre-defined in Custom Instructions)

### Task 2: Prioritization Analysis

**Optimized Prompt Pattern:**
```
Help prioritize these features:
[list 3-5 feature ideas]

Prioritization framework: [RICE/MoSCoW/Value vs Effort]

For each feature, analyze:
- User value (0-10)
- Implementation effort (S/M/L/XL)
- Strategic alignment
- Risks and dependencies

Recommend: Build now / Build later / Don't build
```

**Why this works:**
- Clear decision framework
- Product context from Custom Instructions
- Actionable recommendations
