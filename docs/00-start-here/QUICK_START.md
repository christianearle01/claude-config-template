# Quick Start - 5 Minutes

**For people who hate reading documentation.**

---

## Step 1: Copy CLAUDE.md (2 min)

```bash
cp templates/CLAUDE.md.template ./CLAUDE.md
```

Open `CLAUDE.md` and edit these sections:
- **Line 10-15:** Add your project name and purpose
- **Line 50-60:** List your tech stack (languages, frameworks)
- **Line 100-110:** Add common commands (dev, test, build)

---

## Step 2: Copy Settings (1 min)

```bash
cp -r ~/claude-config-template/.claude ~/your-project/.claude
```

---

## Step 3: Test It (2 min)

Open Claude Code in your project and ask:

> "What does this project do?"

**If Claude answers correctly using info from CLAUDE.md → ✅ Success!**
